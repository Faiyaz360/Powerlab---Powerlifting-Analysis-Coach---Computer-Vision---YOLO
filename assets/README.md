# assets

Drop the header banner here as **`banner.png`** (wide, ~1500×500). When present, the app shows it at
the top of every tab; otherwise it falls back to the built-in SVG wordmark (`BANNER_SVG` in `app.py`).

After adding `banner.png`, deploy it to the Space:

```
hf upload Faiyaz360/Power-Lab assets/banner.png assets/banner.png --repo-type=space
```
